# 🎡 Decision Wheel: Project Report

## 1. Executive Summary
The **Decision Wheel** is a high-performance Single Page Application (SPA) designed to help users make quick, random choices in a fun and interactive way. Built with modern web technologies, it emphasizes **fluid animations**, **responsive design**, and a **premium visual aesthetic**.

## 2. Key Features
- **Dynamic Canvas Wheel**: Automatically adjusts slices based on the number of options with a smooth, physics-based rotation engine.
- **AI Decision Mode**: Pre-defined suggestions for common dilemmas (Food, Activities, Coding Tasks) to get users started instantly.
- **Integrated Sound System**: Synthesized "tick" and "winning" sound effects using the Web Audio API (no external asset dependencies).
- **Interactive Results**: Confetti celebrations and a "Winner" modal with the option to remove the selected choice for multi-round decisions.
- **Persistence**: Uses `localStorage` to ensure options are preserved across browser refreshes.
- **Responsive Fluidity**: Fully functional on both desktop and mobile devices with a glassmorphic sidebar and scrollable lists.

### Audio Engine
Instead of loading heavy `.mp3` files, the app uses a [SoundService](file:///c:/Users/DELL/Desktop/SPA_AJS/src/utils/sounds.js#1-54) to generate sine and square waves on the fly. This results in:
- **Zero latency** sound triggers.
- **Zero bandwidth** usage for audio assets.
- **Dynamic pitch** adjustment (future-ready for variable speed).

## 5. How to Use
1. **Add Options**: Type your choices in the input box and press Enter or the (+) button.
2. **AI Presets**: Use the "AI Suggestions" chips to quickly fill the wheel with themed choices.
3. **Spin**: Click the massive "SPIN THE WHEEL" button.
4. **Decide**: The wheel will slow down and land on a random choice.
5. **Action**: Choose to "Keep" the option or "Remove & Close" to spin again without that choice.
